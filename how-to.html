How to Use This Tool

The Core Idea

This platform takes your HTML and CSS code, renders it using a real browser engine in the background, and captures an image of the result. This is ideal for creating shareable images of UI components, social media posts, email signatures, and more.

The Workflow

1. Choose a Template (Optional)

The best way to start is by selecting an option from the "Start with a template" dropdown. This will automatically fill the code editors for you.

Note: The templates are just starting points! Feel free to modify the HTML and CSS to make them your own.

2. Choose Your Input Mode

You can write your code in two ways. The templates will automatically populate both for you.

  • Separate HTML & CSS: This is the recommended mode for editing, as it keeps your structure (HTML) and styles (CSS) organized and clean. The editors are stacked vertically.
  • Combined HTML with style tag: This mode uses a single editor. Your CSS should be placed inside a tag, followed by your HTML content.

3. Edit Code & See the Live Preview

As you type in the code editors, the Live Preview panel on the right will automatically update. This gives you instant feedback on your changes. Please note the live preview is a lower-quality temporary image generated quickly for speed.

4. Configure Your Final File

Below the main content area, you will find the final output settings:

  • Final Format: Choose between PNG, JPEG, or PDF.
  • Transparent Background (PNG Only): When checked, the final downloaded PNG will have a fully transparent background. This is perfect for placing your image on other backgrounds.
  • Quality (JPEG Only): This slider controls the compression level for JPEG files. Lower quality means a smaller file size.

5. Generate and Download

When your preview looks right and your settings are configured, click the blue "Generate Downloadable File" button. This performs the final, high-quality render.

Info!: The live preview does **not** update for the final format options (like PDF or Transparency). If you change a setting like toggling "Transparent Background", you **must click the "Generate" button again** to apply that change to the final file before downloading or copying.
